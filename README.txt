autovc-master_original : autoVC 모델을 speech accent archive로 실험한 코드입니다.

manipulate_voice.ipynb : .ipynb는 google colab에서 실행해주시기 바랍니다. 기말 발표 데모에서 보여드렸던 목소리 변조 실험 결과입니다. 이 실험의 input 데이터는 english14.wav와 english29.wav 입니다.

manipulate_voicelab.ipynb : 중간 발표 데모에서 보여드렸던 목소리 필터 구현 실험입니다. 

voicelab.ipynb : voicelab 이라는 오픈SW를 이용하여 목소리를 수치화하여 표현한 데이터셋을 제작하는 코드입니다. 

demo.html : 목소리 input을 넣으면 목소리의 속성별 슬라이더로 조정을 하고 튜닝한 목소리를 들려주는 웹입니다. 아직 미완성 단계입니다.
